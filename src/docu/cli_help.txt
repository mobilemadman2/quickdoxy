usage: quickdoxy.py [-h] [--logfile LOGFILE] [--input INPUT] [--output OUTPUT]
                   [--batchmode]

Args that start with '--' (eg. --logfile) can also be set in a config file
(quickdoxy.ini). Config file syntax allows: key=value, flag=true,
stuff=[a,b,c] (for details, see syntax at https://goo.gl/R74nmi). If an arg is
specified in more than one place, then commandline values override config file
values which override defaults.

optional arguments:
  -h, --help         show this help message and exit
  --logfile LOGFILE  logfile name.
  --input INPUT      input files/directories (seperated by ";")
  --output OUTPUT    output directory
  --batchmode        pass this flag to skip the gui
